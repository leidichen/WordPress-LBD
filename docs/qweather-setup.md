# 自定义天气设置指南：如何获取和风天气 API 和城市 ID

为了让您的网站能准确显示当地的天气信息，我们需要配置**和风天气（QWeather）**的 `API Key` 和对应的 `城市 ID`。

请按照以下简单步骤进行获取：

## 步骤一：获取和风天气 API Key

1. **访问官网**：打开 [和风天气开发者控制台](https://console.qweather.com/)。
2. **注册/登录账号**：如果您还没有账号，请先使用邮箱注册并登录。
3. **创建项目**：
   - 登录后，在左侧导航栏点击 **“项目管理”**。
   - 点击右上角的 **“创建项目”**。
   - 填写项目名称（例如：我的网站天气），选择 **“免费订阅”**。
4. **生成 API Key（凭据）**：
   - 在刚刚创建的项目中，点击 **“添加凭据”**。
   - 凭据类型选择 **“Web API”**。
   - 填写凭据名称，点击创建即可。
   - 创建成功后，您会看到一串由字母和数字组成的 **KEY**，这也就是我们在设置中需要填写的 **API Key**。请将其复制并保存好。

> [!TIP]
> 免费订阅的额度对于普通的个人网站展示来说已经完全够用，不需要产生任何费用。

---

## 步骤二：获取城市 ID (Location ID)

和风天气使用一串专用的数字 ID 来定位城市（例如，北京的 ID 是 `101010100`）。获取这款 ID 最简单的方法是通过网页搜索：

1. 打开 [和风天气的主页](https://www.qweather.com/)。
2. 在页面中间或右上角的**搜索框**中，输入您想要显示天气的城市名称或区县名称（比如：“南京” 或 “海淀”）。
3. 在下拉提示框中，选择最准确的那个城市选项并点击。
4. 页面跳转后，请查看浏览器上方的**网址栏（URL）**。
   - 网址通常长这样：`https://www.qweather.com/weather/nanjing-101190101.html`
   - 网址结尾处的一串数字，即 `nanjing-` 后面的 **`101190101`**，这就是您需要的 **城市 ID**。
5. 将这串数字提取出来，填入后台的天气设置中。

---

## 步骤三：完成设置

将上面获取到的 **API Key** 和 **城市 ID** 分别粘贴到后台对应的输入框中，并保存设置。您的网站天气组件就可以正常工作啦！

如果您在操作中遇到任何问题，可以随时参考 [和风天气官方的开发文档](https://dev.qweather.com/)。
