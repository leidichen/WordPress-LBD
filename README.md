# 🐻 LBD (Little Bear Diary) - 小熊日记

> 一款基于 YayU Theme 深度改造的极简 WordPress 主题，灵感源自 Bear Blog。专注于卡片式闪念记录与纯粹的阅读体验。

![License](https://img.shields.io/badge/license-CC%20BY--NC--SA%204.0-green) ![Version](https://img.shields.io/badge/version-1.2.1-blue)

## ✨ 核心特性

- 🌓 **自动明暗模式**：跟随系统设置自动切换，夜晚护眼，白天清晰。
- 🗂️ **卡片式闪念**：像发朋友圈一样记录生活瞬间，支持多图展示和文字折叠。
- 📰 **周刊网格布局**：专为周刊、作品集设计的网格视图，移动端自动适配双栏。
- ⚡ **极致轻量**：移除多余的 WordPress 臃肿功能，无需复杂插件，开箱即用。
- 📱 **完美响应式**：从宽屏到手机，每一处细节都经过精心打磨。

## 🚀 安装方法

1.  下载主题压缩包 `lbd-theme.zip`。
2.  进入 WordPress 后台，点击 **外观** -> **主题** -> **安装主题** -> **上传主题**。
3.  选择下载的压缩包并安装。
4.  点击 **启用**，搞定！🎉

## 📖 使用指南

### 1. 页面模板设置 (关键步骤)

LBD 主题通过**页面模板**来实现不同的布局功能。请按以下步骤创建核心页面：

#### 📝 博客首页 (Blog)
- 新建一个页面，标题任意（如“博客”）。
- 在右侧 **页面属性** -> **模板** 中选择 **博客模板**。
- 发布页面。
- *效果：显示标准的文章列表，极简风格，每页 30 篇。*

#### 💡 闪念/卡片 (Ideas)
- 新建一个页面，标题任意（如“闪念”）。
- 在 **页面属性** -> **模板** 中选择 **卡片模板**。
- 发布页面。
- *效果：显示类似 Twitter/朋友圈的卡片流，每页 20 篇，支持图片画廊。*

#### 📰 周刊/网格 (Weekly)
- 新建一个页面，标题任意（如“周刊”）。
- 在 **页面属性** -> **模板** 中选择 **周刊模板**。
- 发布页面。
- *效果：显示网格状的卡片布局，适合展示封面图，每页 12 篇。*

---

### 2. 个性化配置 (外观 -> 自定义)

进入 **外观** -> **自定义**，你可以调整主题的细节：

#### 🏷️ 站点身份
- 设置你的 **站点标题** 和 **副标题**。
- 上传 **站点图标** (Favicon)。

#### 🃏 卡片设置 (针对闪念页面)
- **关联分类标识**：输入你用于“闪念”的分类 Slug（默认为 `flash` 或 `闪念`）。
- **内容显示字数**：设置卡片折叠前的显示字数。
- **卡片圆角**：调整卡片的圆润程度，随心所欲。

#### 📐 网格设置 (针对周刊页面)
- **网格列数**：选择 2 栏或 3 栏布局（桌面端）。
- **卡片间距**：调整网格之间的空隙大小。

---

### 3. 分类管理小贴士

为了让首页近期文章、闪念和周刊互不干扰，建议：
1.  创建 **闪念** 分类（Slug: `idea`），用于发布类似朋友圈的碎碎念。
2.  创建 **周刊** 分类（Slug: `weekly`），用于发布周刊文章。
3.  主题会自动在首页“近期文章”中**排除**这两个分类的内容，保持首页纯净。

## 🛠️ 高级功能

- **代码高亮**：内置代码块样式，支持水平滚动，适配明暗模式。
- **图片画廊**：在文章中插入多张图片，卡片模式下会自动生成漂亮的拼图效果。
- **极简分页**：全新设计的分页导航，简洁优雅。
